# Novel Navigator (SR-Novel)

在 VS Code 活动栏中浏览和管理本地小说文件夹，支持文本导入、章节预览、快捷键导航等功能。

## 主要特性

- 小说文件夹树视图，支持多文件夹管理
- 支持导入 TXT 文本，自动章节识别
- 章节、行内容预览，支持状态栏/面板两种显示模式
- 快捷键支持：章节/行快速跳转
- 进度自动保存，断点续读
- 右键菜单支持文件/文件夹删除、导入
- 支持自定义显示模式、启用/停用快捷键

## 快速开始

1. 在活动栏点击 **SR-Novel** 图标，打开资源库视图
2. 点击顶部“新建小说文件夹”按钮，创建文件夹
3. 右键文件夹或点击“导入小说文本”导入 TXT/MD 文件
4. 单击文件节点即可开始阅读，支持章节/行导航


## 快捷键

- 下一句：→
- 上一句：←
- 下一章：Ctrl+→
- 上一章：Ctrl+←
- 切换显示模式：Ctrl+Alt+M

> 快捷键需在扩展启用且有选中文本时生效。

## 兼容性

- 需要 VS Code 1.80.0 及以上版本
